# Family Recipes

Weekly family meal plans and consolidated organic shopping lists, auto-generated every Monday morning.

**The diet matrix (every meal works for all four):**
- Vegan core for both adults (one parent vegan; the other Indian vegetarian, dairy OK but no eggs)
- Kid add-ons for the 4- and 1.5-year-old: Wurst, Fisch, eggs
- Optional dairy finish for the vegetarian parent

Preferences: organic fruit & veg, fewer-but-quality products, seasonal (Germany).
